A financial data dashboard

This project was developed as part of my 3-month Data Engineer course at Datascientest, in parallel to the pedagogical cursus (which covers topics from Machine Learning essentials to Kubernetes, via SQL, MongoDB, Scala, Spark,...).

The Github repo's README is the most up-to-date documentation on this project.

You can also see :

  • A live version of the data dashboard, running on a very slow (but free) Oracle Cloud instance
  • A live version of the API documentation (use "guest"/"guest" as username/password)
  • A PDF report
  • A presentation in slides

The technical stack consists of :

Dashboard
Dash
API server
FastAPI
Stock market data storage
MongoDB
Deployment
Docker
Utility libraries
  • Pydantic (data validation)
  • Pandas (data processing)
  • Pytest (unit/integration/e2e testing)
  • Dynaconf (configuration management)
